# Usage

## Start CARLA

Launch the CARLA simulator before running any scripts.

## Train DDQN Agent

```bash
python scripts/train_ddqn.py
```

## Evaluate Agent

```bash
python scripts/evaluate.py
```

## Run Benchmark

```bash
python scripts/run_benchmark.py
```

## Expected Outputs

Results are stored in:

```text
results/
├── logs/
├── csv/
└── figures/
```

## Model Checkpoints

Store trained models inside:

```text
models/
```
